# Quick Start Guide

## Installation & Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   - Navigate to `http://localhost:3000`
   - The app will open automatically

## First Steps

### 1. Add Your First Company
- Click "Add Company" button
- Enter company name (e.g., "Acme Corp")
- Click "Add"

### 2. Add Departments
- Click on the company card to open it
- Click the "+" button in the Departments sidebar
- Add departments like "Tech", "Marketing", etc.
- Or use predefined departments

### 3. Add Tasks
- Select a department from the sidebar
- Click "Add Task"
- Fill in:
  - Task Name (required)
  - Assigned To (optional)
  - Deadline Date (required)
  - Status (defaults to "Not Started")
- Click "Add Task"

### 4. View Dashboard
- Click "Dashboard" in the navigation
- See overview of all tasks
- Filter by: Overdue, Due Today, Upcoming, Completed

- All data persists between sessions
- Delete tasks using the trash icon on task cards
- Delete companies from the company list view
- View company statistics in the Dashboard
